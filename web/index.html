나노리본 경계 상태 (Nanoribbon Analysis)

벌크 평탄 밴드로부터 나노리본의 경계 상태(Edge Mode)를 수치적으로 분석합니다. (2D 전용)

나노리본 에너지 분산 (Dispersion)

점 선택 시 우측에 실공간 파동함수가 표시됩니다.

파동함수 실공간 분포 (Wavefunction Profile)

분산 플롯에서 상태(점)를 선택해 주세요.
나노리본 실공간 국소화 패턴 (원 크기/색상 = 확률 밀도)

강건한 경계 모드 (Robust Boundary Mode)

토러스에서 정확히 소멸하는 특이(singular) 평탄 밴드 CLS의 합이, 열린 경계(OBC)에서는 경계에만 남는 현상을 계산·시각화합니다. (Rhim & Yang)
시스템 크기 N_x N_y 대상 평탄 밴드 index 결함 셀 (선택)

실공간 경계 모드 진폭 |ψ_∂(R)|

마커 크기 ∝ |ψ|, 색상 = 부호/위상. 내부(Bulk)는 완전히 비어 있어야 함.

Skin Depth 프로파일

경계로부터의 거리별 최대 진폭. 지수감쇠가 아닌 Step-function이어야 함.

실공간 CLS 기반 Topological Flat Band 설계 (Engineering)

실공간 격자에서 CLS(Compact Localized State)를 직접 배치하거나(수동 배치), 원하는 Chern 수와 브릴루앙 영역 특이점 조건을 지정하여(자동 탐색) 역공학으로 평탄 밴드 해밀토니안을 설계합니다. (2D 전용)

🔲 실공간 격자 미리보기 (중심 셀 + 이웃 셀)

자동 탐색: 목표 Chern 수 C와 브릴루앙 영역 특이점을 지정하면 시스템이 CLS를 자동 설계합니다. · 수동 배치: 위 격자에서 사이트를 클릭해 CLS에 포함/제외하고(기본값 A=1, θ=0), 왼쪽 표에서 진폭/위상을 조정하세요.
